Cloud, DevOps, Kubernetes & AI Engineering Portfolio
Overview

This repository contains a collection of hands-on cloud, DevOps, Kubernetes, AI, and software engineering projects developed to demonstrate real-world infrastructure, automation, deployment, monitoring, security, and application development skills.

The projects simulate enterprise environments and showcase technologies commonly used in modern cloud engineering, platform engineering, site reliability engineering (SRE), DevOps, and AI application development.
